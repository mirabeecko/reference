# React demo weby

V této složce je 8 samostatných React projektů postavených na Vite:

- `01-remeslnik`
- `02-bistro`
- `03-konzultant`
- `04-apartman`
- `05-advokat`
- `06-fitness`
- `07-produkt`
- `08-makler`

Každý projekt spustíš samostatně:

```bash
cd 01-remeslnik
npm install
npm run dev
```

Stejný postup platí pro ostatní složky.
